### Info
Repository ini berisi aplikasi perpustakaan (inventory dan peminjaman). Aplikasi ini dibuat menggunakan Laravel untuk tugas akhir kuliah.

Komponen yang digunakan :
- Laravel 6.9.0
- Bootstrap 4.4
- Fontawesome 4.7.0
- Datatable
- jQuery
